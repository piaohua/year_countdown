# 今年天数倒计时

一个 `Flutter` `Demo`:

## 使用

如果有适配问题或者其他需求可以使用源码重新编译: 

* 下载源码

``` bash
git clone https://github.com/piaohua/year_countdown.git
```

* 删除 `build`, `ios`, `android`, `test` 文件

* 使用 `Flutter` 初始化项目

``` bash
flutter init

flutter run -d chrome
```
